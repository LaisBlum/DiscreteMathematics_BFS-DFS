# DiscreteMathematics_BFS-DFS
Project work of the discipline Discrete Mathematics

Consider an arbitrary graph with at least 15 vertices. Write programs that implement the 
Depth-First Search (DFS) and Breadth-First Search (BFS) algorithms to find a spanning tree
of this graph. Draw a graph and its spanning trees (one from DFS and one from BFS).

• Find an example where both algorithms result in the same spanning tree.
• Find the number of possible spanning trees that can be produced by DFS and BFS, or
write a program that produces all possible spanning trees.
• How do DFS and BFS look for different types of graphs, e.g., star, complete, bipartite, or
Turán graphs?
